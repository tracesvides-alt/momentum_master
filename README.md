# 🚀 Momentum Master (モメンタム・マスター)

**Momentum Master** は、米国株市場における「真の強者」を見つけ出すための Streamlit アプリケーションです。
単なる騰落率ランキングではなく、**ミネルヴィニ流の成長株投資基準（Trend Templates）** や **レラティブ・ストレングス（RS）** を考慮し、持続的な上昇トレンドにある銘柄を厳選します。

さらに、AIを活用した**ニュース分析・要約機能**を搭載しており、「なぜその銘柄が上がっているのか？」を即座に日本語で把握できます。

---

## ✨ 主な機能

### 1. 🏆 モメンタム・ランキング (Top 10 / Worst 10)
数千の米国株の中から、以下の基準でスクリーニングを行います。
- **RS Rating (レラティブ・ストレングス)**: 市場の90%以上の銘柄より強いか？
- **Trend Templates**:
    - 株価 > SMA50 > SMA150 > SMA200
    - 52週高値から25%以内にあるか
    - 出来高の急増（Volume Spike）
- **除外基準**: ペニーストック、超低流動性銘柄、買収予定銘柄などを自動除外

### 2. 🚨 Opportunity Alert (チャンス通知)
起動時に以下の「買い合図」を自動検知してトップ画面に表示します。
- **Persistence (持続性)**: 選択した期間で「3日連続」トップ10入りしている。
- **Volume Spike (出来高急増)**: 直近の出来高が過去平均の **2倍以上** に急増している。

### 3. 📰 AI ニュース分析 ("Why is it moving?")
Yahoo Finance等のニュースを取得し、以下の処理を行います。
- **ノイズ除去**: "Options", "Prediction", "Why it is moving" などの薄い自動生成記事をフィルタリング。
- **カタリスト優先**: "Earnings"（決算）, "Deal"（契約）, "FDA/Launch"（発表）などの重要ニュースを優先表示。
- **Deep AI要約**: 記事本文をAIが解析し、**宣伝文言（「無料レポート」「他の推奨株」）を排除** した上で、日本語で要約を表示します。

### 4. 🌡️ セクターヒートマップ & テーマ分析
- 主要セクター（Tech, Semis, Software, Energy等）の強弱をヒートマップで可視化。
- 「AI」「Cybersecurity」「Bitcoin」など、特定のテーマ株のモメンタムも追跡可能。

## 📂 ファイル構成

- `momentum_master_app.py`: メインのアプリケーションファイル (UI, ニュースロジック)
- `market_logic.py`: 株価計算、スクリーニング、テクニカル分析のコアロジック
- `data/`: 株価データのキャッシュフォルダ (Parquet形式)
- `requirements.txt`: 必要なPythonライブラリ一覧
